# ConCurBot :rocket:

## Бот, который конвертирует выбранную валюту в рубли.<h1/>

* Быстрый старт:

```python
    source ./.venv/bin/activate
    pip install -r requirements.txt
    python bot.py
```

* Использование:

  ```commandline
  /convert {количество валюты} {код валюты (регистр не важен)}

  возвращает конвертируемую сумму в рублях
  ```
